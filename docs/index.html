Quick Start

Get Started in Minutes

Install, import, and start creating beautiful charts with just a few lines of code.

üì¶ Installation

Install via npm:

npm install your-chart-library

Or via CDN:

<script src="https://cdn.example.com/charts.js"></script>

üíª Basic Usage

Create your first chart:

import { BarChart } from 'your-chart-library';

const data = [
  { name: 'Jan', value: 400 },
  { name: 'Feb', value: 300 },
  { name: 'Mar', value: 600 }
];

<BarChart 
  data={data}
  x-key="name"
  y-key="value"
  color="#3b82f6"
/>
